

🚀 Full-Stack Ecommerce Practice

This is a practice project developed to master the complete workflow of a web application, from creating a REST API with .NET to building a dynamic user interface with Bootstrap.

🛠️ Technologies Used

Backend

C# / .NET 9: Server-side logic and API development.
Entity Framework Core: ORM for data management.
SQLite: Persistent relational database.
Swagger/OpenAPI: API documentation and testing.

Frontend

HTML5 / JavaScript (ES6+): Client-side logic and API consumption (Fetch API).
Bootstrap 5: Responsive design framework and UI components (Modals, Toasts, Cards).
UX/UI Principles: Implementation of empty states, visual feedback (notifications), and adaptive design.

✨ Key Features

Full CRUD: Create, Read, Update, and Delete products.
Real Persistence: Data is stored in a local SQLite database.
Real-Time Search: Product filtering via JavaScript without page reloads.
Security Validations: Server-side data protection using Data Annotations.
Responsive Interface: Designed to display correctly on both mobile and desktop devices.

🚀 Full-Stack Ecommerce Practice

Este es un proyecto de práctica desarrollado para dominar el flujo completo de una aplicación web, desde la creación de una API REST con .NET hasta una interfaz de usuario dinámica con Bootstrap.

🛠️ Tecnologías Utilizadas

Backend
-C# / .NET 9: Lógica de servidor y API.
-Entity Framework Core: ORM para la gestión de datos.
-SQLite: Base de datos relacional persistente.
-Swagger/OpenAPI: Documentación y pruebas de la API.

Frontend
-HTML5 / JavaScript (ES6+): Lógica del lado del cliente y consumo de API (Fetch API).
-Bootstrap 5: Framework de diseño responsivo y componentes UI (Modales, Toasts, Cards).
-UX/UI Principles: Implementación de estados vacíos, feedback visual (notificaciones) y diseño adaptable.

✨ Características Principales

-CRUD Completo: Crear, Leer, Editar y Eliminar productos.
Persistencia Real: Los datos se guardan en una base de datos SQLite local.
Buscador en Tiempo Real: Filtrado de productos mediante JavaScript sin recargar la página.
-Validaciones de Seguridad: Protección de datos en el servidor (Data Annotations).
-Interfaz Responsiva: Diseñada para visualizarse correctamente en móviles y escritorio.
