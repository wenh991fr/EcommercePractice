🚀 Full-Stack Ecommerce Practice

Este es un proyecto de práctica desarrollado para dominar el flujo completo de una aplicación web, desde la creación de una API REST con .NET hasta una interfaz de usuario dinámica con Bootstrap.

🛠️ Tecnologías Utilizadas

Backend
-C# / .NET 9: Lógica de servidor y API.
-Entity Framework Core: ORM para la gestión de datos.
-SQLite: Base de datos relacional persistente.
-Swagger/OpenAPI: Documentación y pruebas de la API.

Frontend
-HTML5 / JavaScript (ES6+): Lógica del lado del cliente y consumo de API (Fetch API).
-Bootstrap 5: Framework de diseño responsivo y componentes UI (Modales, Toasts, Cards).
-UX/UI Principles: Implementación de estados vacíos, feedback visual (notificaciones) y diseño adaptable.

✨ Características Principales

-CRUD Completo: Crear, Leer, Editar y Eliminar productos.
Persistencia Real: Los datos se guardan en una base de datos SQLite local.
Buscador en Tiempo Real: Filtrado de productos mediante JavaScript sin recargar la página.
-Validaciones de Seguridad: Protección de datos en el servidor (Data Annotations).
-Interfaz Responsiva: Diseñada para visualizarse correctamente en móviles y escritorio.

-Abre index.html en tu navegador (recomendado usar la extensión Live Server en VS Code).
